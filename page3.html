القوائم (Lists)

تُستخدم القوائم لتنظيم العناصر في شكل منظم ومرتب. هناك نوعان رئيسيان:

<ul> - القائمة غير المرتبة

عرض العناصر بنقاط بدون ترتيب معين:

<ul>
    <li>عنصر 1</li>
    <li>عنصر 2</li>
    <li>عنصر 3</li>
</ul>

<ol> - القائمة المرتبة

عرض العناصر بأرقام أو حروف مرتبة:

<ol>
    <li>الخطوة الأولى</li>
    <li>الخطوة الثانية</li>
    <li>الخطوة الثالثة</li>
</ol>

<li> - عنصر القائمة

يُستخدم لتعريف العناصر داخل أي نوع قائمة:

<ul>
    <li>تفاح</li>
    <li>موز</li>
    <li>برتقال</li>
</ul>

أمثلة عملية

قائمة تسوق غير مرتبة:

<ul>
    <li>خبز</li>
    <li>حليب</li>
    <li>بيض</li>
</ul>

قائمة مهام مرتبة:

<ol>
    <li>فتح المتصفح</li>
    <li>كتابة الكود</li>
    <li>حفظ الملف</li>
</ol>
